✨🌟 **TP React – Navigation, Composants et Médias** 🌟✨

────────────────────────────
🎯 **Objectif**
────────────────────────────

Consolider les concepts fondamentaux de **React** :

* Création de **composants fonctionnels**
* Navigation avec **React Router**
* Gestion d’état avec **useState**
* Rendu conditionnel
* Affichage dynamique avec `.map()`
* Intégration d’images et de médias
* Application de styles CSS

────────────────────────────
🧩 **Composants**
────────────────────────────

🌸 **Accueil** : page principale contenant les composants de l’application.

🌸 **Apropos** : présente l’objectif de l’application et explique son fonctionnement.

🌸 **Contact** : affiche les informations de contact.

🌸 **Connexion** : exemple de rendu conditionnel utilisant `useState` pour afficher l’état connecté/déconnecté.

🌸 **ListeTaches** : affiche une liste dynamique de tâches avec la méthode `map()`.

🌸 **Galerie** : affiche plusieurs images importées depuis le dossier `src`.

🌸 **LecteurAudio** : lecteur audio utilisant la balise HTML `<audio>`.

────────────────────────────
💡 **Concepts clés**
────────────────────────────

🔹 **Composants React** : blocs réutilisables pour structurer l’application

🔹 **React Router** : navigation entre plusieurs pages (`Accueil`, `À propos`, `Contact`)

🔹 **useState** : gestion de l’état dans un composant

🔹 **Rendu conditionnel** : affichage différent selon une condition

🔹 **map()** : génération dynamique d’éléments à partir d’un tableau

🔹 **Import des ressources** : images et fichiers audio intégrés dans React

🔹 **CSS** : stylisation de l’interface utilisateur

────────────────────────────
🚀 **Exécution du projet**
────────────────────────────

1️⃣ Vérifier que **Node.js** et **npm** sont installés

2️⃣ Installer les dépendances :

```
npm install
```

3️⃣ Lancer l’application :

```
npm start
```

4️⃣ Ouvrir dans le navigateur :

```
http://localhost:3000
```
